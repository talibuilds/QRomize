## ✨ Features

* 🎨 **Custom Colors**: Change dots, background, and marker colors.
* ⚡ **Live Preview**: See changes instantly with zero lag.
* 🏢 **Logo Upload**: Add your brand logo inside the QR code.
* 🔤 **Text & Emojis**: Use text or emojis as central icons.
* 🕸️ **Interactive UI**: Mouse-reactive spidernet background.
* 📦 **Export**: Download as SVG or PNG.

## 🚀 Quick Start

```bash
git clone https://github.com/talibuilds/QRomize.git
cd QRomize
npm install
npm run dev
```

Open `http://localhost:5173`.

## 🛠️ Tech Stack

* **React.js**
* **Vite**
* **react-router-dom**
* **qr-code-styling**
* **lucide-react**
* **Vanilla CSS**
